жҹҘзңӢжҹҗдёҖз«ҜеҸЈеҚ з”Ёжғ…еҶө

з¬¬дёҖз§Қж–№жі•пјҡдҪҝз”Ё netstat

sudo netstat -ltpn | grep :22

з¬¬дәҢз§Қж–№жі•пјҡдҪҝз”Ё lsof

еӣ дёәеңЁ linux дёҠдёҖеҲҮзҡҶж–Үд»¶пјҢTCP socket иҝһжҺҘд№ҹжҳҜдёҖдёӘ fdгҖӮеӣ жӯӨдҪҝз”Ё lsof д№ҹеҸҜд»Ҙ
sudo lsof -n -P -i:22

#-n иЎЁзӨәдёҚе°Ҷ IP иҪ¬жҚўдёә hostnameпјҢ
#-P иЎЁзӨәдёҚе°Ҷ port number иҪ¬жҚўдёә service name
#-i:port иЎЁзӨәз«ҜеҸЈеҸ·дёә 22 зҡ„иҝӣзЁӢ

еҰӮдҪ•жҹҘзңӢиҝӣзЁӢзӣ‘еҗ¬зҡ„з«ҜеҸЈеҸ·

з¬¬дёҖз§Қж–№жі•пјҡдҪҝз”Ё netstat

sudo netstat -atpn | grep 1333

з¬¬дәҢз§Қж–№жі•пјҡдҪҝз”Ё lsof

sudo lsof -n -P -p 1333 | grep TCP
